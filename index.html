Enterprise Network Design Project

Project By: Muhammad Basil, Aiswarya Santhosh, Anand Manoharan, Devarsh M D, Nived G Unni

Protocols: TCP, SHRSLS, RPHTPMLA, OSPF

Location A - Internal Network

  • Devices: PCs (PC0–PC9), Access Points, Smartphones, Laptops, Tablets
  • VLANs: VLAN 10 (172.16.1.0/24 - Wired), VLAN 20 (172.16.2.0/24 - Wireless)
  • Security: Host-Based Firewall on all PCs

Core Network - Routed Backbone

  • Routers: Router0 to Router5
  • Subnets: 10.2.1.0/24, 10.16.1.0/24, 172.24.1.0/24, 192.168.150.0/24, 192.168.190.0/24, 192.168.200.0/24
  • Firewall: Network-Based Firewall between Router3 and Router4

Location B - Corporate Network

  • Devices: PCs (PC10–PC18), MAN-SERVER, Server0, WLC, Lightweight AP
  • VLANs: VLAN 90 (172.18.90.0/24 - Wireless), VLAN 91 (172.18.51.0/24 - Wired)
  • Wireless: Controlled by Wireless LAN Controller

Simulation Features

  • Ping, Web Access, DNS and DHCP Testing
  • Firewall Filtering and ACL Scenarios
  • Inter-VLAN Routing and Wireless Client Behavior
